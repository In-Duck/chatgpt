# Project Summary
이 프로젝트는 Windows 환경에서 PyQt5를 사용하여 키 입력 자동화 및 창 모니터링 기능을 제공하는 애플리케이션입니다. 사용자는 특정 창을 선택하고, 해당 창에서 자동으로 키 입력을 수행하도록 설정할 수 있습니다. 이 제품은 반복적인 작업을 자동화하고자 하는 사용자에게 유용하며, 사용자 감지 및 UI 구성 요소가 개선되었습니다. 최근에는 버프 기능이 추가되어 사용자가 설정한 키를 자동으로 입력할 수 있는 기능이 포함되었습니다.

# Project Module Description
- **main.py**: 애플리케이션의 진입점.
- **main_window.py**: 주 창 UI를 정의하며, 사용자 인터페이스와 기능을 연결합니다.
- **settings_dialog.py**: 설정 대화 상자를 제공하여 사용자가 환경설정을 조정할 수 있습니다.
- **window_monitor.py**: 현재 활성화된 창을 모니터링하고, 특정 창이 활성화되면 신호를 발생시킵니다.
- **key_input_worker.py**: 자동 키 입력을 처리하며, 설정된 간격에 따라 키를 누릅니다.
- **user_detector.py**: 사용자 감지를 위한 모듈입니다.
- **region_preview.py**: 감지할 영역을 미리보기 위한 모듈입니다.
- **buff_worker.py**: 추가된 버프 기능을 처리합니다.
- **config_manager.py**: 설정 파일을 로드하고 저장하는 기능을 제공합니다.
- **requirements.txt**: 프로젝트의 의존성 목록입니다.
- **todo.md**: 프로젝트의 향후 계획 및 할 일 목록입니다.

# Technology Stack
- **PyQt5**: GUI 애플리케이션을 위한 프레임워크.
- **pywin32**: Windows API와 상호작용하기 위한 라이브러리.
- **pynput**: 키 입력을 제어하기 위한 라이브러리.
- **Pillow**: 이미지 처리 라이브러리.
- **pyautogui**: 자동화 및 GUI 조작을 위한 라이브러리.
- **python-telegram-bot**: 텔레그램 봇 API를 위한 라이브러리.
- **JSON**: 설정 파일 형식으로 사용됩니다.

# Usage
1. 의존성 설치:
   ```bash
   pip install -r requirements.txt
   ```
2. 애플리케이션 실행:
   ```bash
   python main.py
   ```
